# Kiriox – Network Risk Engine

Official system name: `Kiriox – Network Risk Engine`.

## Naming Source Of Truth

The application name is fixed in code and must be maintained in:

- `src/config/app.ts`

Current constants:

- `APP_NAME = "Kiriox – Network Risk Engine"`
- `APP_SHORT_NAME = "Kiriox"`

Do not use database or `.env` as the primary source for the system name.
